# 2024_Social-Welfare-Transportation-Support-System-Using-Deep-Learning
본 프로젝트는 **2024년도 교내 캡스톤 디자인 경진대회**를 위해 진행되었습니다.
  
<br>
<br>
<br>

## 작품 설명
딥러닝 기반의 YOLO 알고리즘을 활용하여 교통약자를 실시간으로 인식하고, 보행 신호를 자동으로 조정하여 안전성과 교통 효율성을 높이는 시스템 개발
